# Language Learning App

Односторінковий SPA-застосунок для вивчення іноземних слів, розроблений на React у рамках фінального проєкту курсу.

## Мета проєкту
Закріпити практичні навички роботи з React, зокрема:
- компоненти та props
- маршрутизація (SPA)
- робота зі станом
- форми та події
- збереження даних
- базове тестування
- архітектура застосунку

---

## Функціонал
- додавання нових слів (слово, переклад, приклад)
- відображення списку слів
- збереження даних у LocalStorage
- умовний рендеринг (порожній список)
- навігація між сторінками (React Router)
- базова стилізація інтерфейсу
- тестування компонентів (Jest, React Testing Library)

---

## Використані технології
- React
- React Router
- JavaScript (ES6+)
- LocalStorage
- Jest
- React Testing Library
- CSS
